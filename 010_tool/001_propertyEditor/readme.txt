安装步骤:
	下载完毕jp.gr.java_conf.ussiy.app.propedit_5.3.3.zip后
	在eclipse 安装目录 links文件夹下 下创建 propertyEditor.link(可以多个.link文件)
		path=mypluins/propertyEditor
	在eclipse 安装目录 mypluins文件夹下 下创建 propertyEditor文件夹
	在propertyEditor文件夹下,放入5.3.3.zip中的 features 和 plugins
